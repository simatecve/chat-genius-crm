---
description: Start the WhatsApp Orchestrator Service
---
This workflow starts the WhatsApp Orchestrator service required for QR code generation and WhatsApp session management.

1. Navigate to the whatsapp directory
```bash
cd whatsapp
```

2. Install dependencies (if not already installed)
```bash
npm install
```

3. Start the server
```bash
npm start
```

The server should start on port 3001.
